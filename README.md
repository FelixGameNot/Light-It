Руководство по пользованию.
Сцена с игрой расположена по пути Assets/Scenes/SampleScene
Создание обьектов для того, чтобы создавать обьекты необходимо навестись на землю и зажать клавишу "С" + ["L"(для создания лампы), или "T"(для создания бочки), или "B"(для создания коробки)].
Для сохранения сцены необходимо нажать кнопку Save, сцена сохранит расположение и настройки всех обьектов на ней в том числе и положение персонажа.
Load Defaults загружает стандартные настройки сцены.
Load загружает ваши сохранения если они существуют, в ином случае дефолтные.
При старте сцены автоматически происходит Load.
